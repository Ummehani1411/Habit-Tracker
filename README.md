# 📖 Overview
The app provides:

- ✅ Daily habit tracking
- 📊 Progress monitoring
- ⏰ Scheduled routines
- 🎯 Goal consistency
- 📱 Responsive UI
- 🌙 Clean modern design

Designed with a warm and minimal user experience to encourage productivity and self-improvement.

# 📸 Core Functionalities

## 📝 Add New Habit

Users can:
- Enter habit name
- Set optional time
- Choose category

## ✔️ Mark Habits Complete

One-click completion system with:
- Animated checkbox
- Progress updates
- Visual feedback

## 🗑️ Delete Habits

Remove habits instantly from the tracker.

---

# 📈 Progress System

The app automatically calculates:

```text
Completed Habits / Total Habits
```

and displays:
- Percentage completed
- Remaining tasks
- Daily productivity stats
